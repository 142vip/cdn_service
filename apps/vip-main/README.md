# vip-main

142vip 主站静态资源，按生活场景分类存放图片。

## 目录说明

| 目录 | 说明 |
| --- | --- |
| [travel](./travel) | 旅游出行 |
| [sports](./sports) | 运动健身 |
| [cooking](./cooking) | 做菜美食 |
| [fishing](./fishing) | 钓鱼休闲 |
| [daily](./daily) | 日常生活 |

## 规范

- 位图优先 `.webp`，图标用 `.svg`
- 文件名 kebab-case，禁止中文
- 单文件 ≤ 2MB
- 新增或变更文件请同步更新对应目录 README
